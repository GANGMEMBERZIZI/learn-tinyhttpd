# Learn-TinyHTTPd

这是一个基于 C 语言实现的轻量级 Web 服务器，作为学习网络编程和 Linux 系统调用的实验项目。

## 功能特性
- [x] 支持静态 HTML 页面访问。
- [x] 支持 GET/POST 请求解析。
- [x] 实现 CGI 交互功能（通过管道通信）。
- [x] 成功在 WSL2 环境下运行并完成 Perl/Shell 脚本调试。
